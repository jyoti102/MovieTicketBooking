# How to Run the Project

## 1. Start Discovery Server

cd discovery-server
mvn spring-boot:run

Access:
http://localhost:8761

---

## 2. Start Admin Panel

cd admin-panel
mvn spring-boot:run

---

## 3. Start Booking System

cd booking-system
mvn spring-boot:run

---

## 4. Start Auth Server

cd auth-server
mvn spring-boot:run

---

## 5. Start API Gateway

cd api-gateway
mvn spring-boot:run

Access Gateway:
http://localhost:8080

---

## Important

Always start Discovery Server first.
